Git es un sistema de control de versiones distribuido que se utiliza para gestionar y registrar los cambios en el código fuente de un proyecto.

Permite a los desarrolladores:

Mantener un historial completo de modificaciones.

Trabajar en paralelo mediante ramas (branches) sin afectar la versión principal.

Colaborar con otros desarrolladores de forma segura.

Revertir cambios si se detectan errores.

En términos prácticos, Git facilita el desarrollo colaborativo, el control de versiones y la trazabilidad del código en proyectos de software.
